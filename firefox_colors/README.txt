|HOW TO IMPORT|
+-------------+

To import this on Firefox:
    about:config > toolkit.legacyUserProfileCustomizations.stylesheets > true

Browse to current profile:
    (Inside firefox) type about:support > Folder Named like: 12abcde3.default-release-1 (make sure it is the active one)> Open Directory
    Create "chrome" Folder
    Export the userChrome.css file into chrome folder (rename it userChrome.css)
